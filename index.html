Markdown

An easy-to-read, easy-to-write plain text format

Posted on Dec 29 2016 · markdown

Markdown is mostly used as a text-to-HTML conversion tool for web writers. After writing using plain text format, your markdown document can be converted to structurally valid XHTML (or HTML). For instance, github uses markdown to preview read me’s. Also the posts on this blog are written in markdown.
